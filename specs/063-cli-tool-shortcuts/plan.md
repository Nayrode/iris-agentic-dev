# Implementation Plan: CLI Tool Shortcuts

**Branch**: `063-cli-tool-shortcuts` | **Date**: 2026-07-11 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/063-cli-tool-shortcuts/spec.md`

## Summary

Add `exec`, `query`, `doc`, and `tool` subcommands to the `iris-agentic-dev` binary (alongside the existing `compile` subcommand), each calling the same underlying IRIS connection and tool logic the MCP server uses. A shared `ConnectionArgs` clap group handles `--namespace`, `--container`, `--host`, `--port`, `--username`, `--password` uniformly. The generic `tool <name> --args <json>` fallback covers the full set of MCP tools via a static dispatch map.

## Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| `tool` dispatch map drifts from `registered_tool_names()` | T025 asserts parity at test time |
| `exec` write-gate allows destructive operations | Reuse `is_write_allowed()` — same gate as MCP `iris_execute` |
| TSV output with embedded tabs in IRIS data | Escape embedded tabs as `\t` literal in output cells |
| Stdin `-` conflicts with clap argument parsing | Use `clap::value_parser` with explicit `-` sentinel handling |
